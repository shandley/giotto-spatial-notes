What is Giotto?

An R-native toolkit (with emerging Python bindings) for comprehensive analysis and visualization of spatial multi-omics data. Handles data across resolutions — subcellular, single-cell, multi-cell — in 2D and 3D.

GiottoClass GiottoUtils GiottoVisuals

Design Philosophy

Technology-agnostic framework generalizable to virtually all current and emerging spatial platforms, regardless of resolution, modality (RNA, protein, DNA), or spatial dimension.

Key Strengths

Dedicated constructors for major platforms, spatial network-aware statistics, image-backed large dataset support via terra/SpatRaster, and solid interop with Seurat and SingleCellExperiment.

Things to Know First

Spatial network construction (Delaunay or kNN by distance) is a prerequisite for most spatial statistics. Build the network before running spatial variable gene tests or proximity enrichment.

◈
Spatial Variable Genes
SpatialDE · SPARK · binSpect
Identifies genes whose expression varies significantly across space. Giotto wraps SpatialDE and SPARK, and includes its own binSpect method which uses the spatial network to test expression gradients.
◉
Cell-Type Deconvolution
RCTD · CARD · Spotlight
Estimates cell-type composition at each spot (for low-resolution data like Visium). Integrates with RCTD, CARD, and Spotlight using a scRNA-seq reference. Results are stored as cell metadata and can be visualized spatially.
⬡
Spatial Co-expression
rankEnrich · page_enrich · HMRF
Detects genes and gene modules with correlated spatial expression patterns. HMRF (Hidden Markov Random Field) models spatially coherent domains by combining expression and spatial context.
⬥
Cell-Proximity Enrichment
cellProximityEnrichment · ICF
Tests whether specific cell-type pairs appear in proximity more or less often than expected by chance. Requires a spatial network. Useful for identifying physical cell-cell interaction niches.
▣
Niche Clustering
calculateSpatialEnrichment · niche
Clusters cells based on the composition of their local neighborhood — combining a cell's own expression with information about surrounding cell types. Captures microenvironment context beyond single-cell clustering.
◌
Dimensionality Reduction
runPCA · runUMAP · spatialPCA
Standard PCA and UMAP are available, plus spatially-aware alternatives that incorporate coordinate information into the embedding. Results integrate with standard Giotto visualization functions.

Spatial network construction

# Delaunay triangulation (topology-based)
createSpatialNetwork(gobject = giotto_obj, method = "Delaunay")

# k-nearest neighbors by distance
createSpatialNetwork(gobject = giotto_obj, method = "kNN", k = 6)
Important: spatial network construction must happen before running spatial statistics — binSpect, cellProximityEnrichment, HMRF, and niche clustering all require it. Build the network immediately after normalization and before any spatial analysis step.

Large dataset handling

For high-resolution datasets (Xenium, Visium HD, MERSCOPE) with millions of transcripts, Giotto uses terra / SpatRaster image-backed objects. These keep data on disk rather than in RAM, enabling analysis of tissue sections that would otherwise exhaust memory. The interface is the same; the backing changes transparently.
